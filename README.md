
# express-mysql-crud

 A basic crud api using express mysql(without orm)



## Installation

yarn is used here
```bash
  cd project
  yarn
  node server.js
```
Server Port & Db Credentials are stored in .env file.Check .env.example for more details.
```bash
SERVER_PORT=

DB_CONNECTION=
DB_HOST=
DB_PORT=
DB_DATABASE=
DB_PASSWORD=
DB_USERNAME=
  
```
